# Este es mi repositorio de mi Juego de piedra, papel o tijeras.

Este juego fue creado en HTML y JavaScript.
para jugar es muy simple una ves tengas abierto el juego escoje entre piedra, papel o tijera.
La computadora va a escojer aleatoria aleatoriamenta.
abajo estan los resultados del la opcion que escojio el usuario y la computadora para determinar el ganador o si fue empate.
si vamos mas abajo tienes un registro del las veses que ganaste, perdiste o fue empate tiene un boton que reinicia el registro.
